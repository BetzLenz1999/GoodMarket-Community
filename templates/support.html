Support desk

How can we help?

Send a request to the GoodMarket team. We securely forward it to the GIMT Support Desk so the existing support dashboard can track and manage your ticket.

🔒Your browser submits only to GoodMarket. Private support-desk secrets stay on the server.
🎫On success, you will receive a support ticket number like SUP-YYYYMMDD-XXXX.
🧾Add an order or transaction ID when the issue is related to a payment, delivery, refund, or trade.
